Pisedof by albertramsbottom

Label Printing Software for Pay Pal

The idea behind this is to create some software in the cloud that allows users to register foe an account. Once they have an account they can give the software permission to access their Pay Pals API. In doing so will allow the software to detect a sale via eBay or any other Pal Pal payment and send an integrated label sheet straight to the printer to print both address label and packing slips This project might expand as in to clear, colorful and concise ebay accounting software as well
